# System Metrics Collector

Сервис для сбора и отображения системных метрик (CPU, память, диск, сеть, процессы) с веб-интерфейсом.

## Возможности

- Сбор метрик в реальном времени
- Веб-дашборд с графиками и таблицами
- Graceful shutdown
- Конфигурация через переменные окружения
- Docker-контейнеризация

## Запуск

### Локально

```bash
go run cmd/server/main.go
